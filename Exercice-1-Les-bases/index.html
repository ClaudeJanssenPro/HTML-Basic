Les bases de l'html

  • - Les balises

    • - Les balises paires

      Une balise paire est en fait composée d'une balise ouvrante et d'une balise fermante, et encadre du texte qu'on veut afficher.
      Par exemple :

      • <html></html>
      • <p></p>
      • <strong></strong>
      • <h1></h1>
      • <b></b>

    • - Les balises orphelines

      self closing tags

      Une balise orpheline est une balise auto-fermante (self closing tag), c'est à dire qui ne nécessite pas une une balise fermante, il est néanmoins recommandé d'inclure un slash (/) dans la balise afin de les repérer aisément. Ce type de balise est placé précisement là où l'on veut afficher l'élément.
      Par exemple :

      • <img />
      • <br />
      • <div />

      (Source : Votre première page web en HTML - OpenClassrooms)
