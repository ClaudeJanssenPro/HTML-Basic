Les bases de l'HTML

  • - Les balises

    • - Les balises paires

      Une balise paire est en fait composée d'une balise ouvrante et d'une balise fermante, et encadre du texte qu'on veut afficher.
      Par exemple :

      • <html></html>
      • <p></p>
      • <strong></strong>
      • <h1></h1>
      • <b></b>

    • - Les balises orphelines

      Une balise orpheline est une balise auto-fermante (self closing tag), c'est à dire qui ne nécessite pas une une balise fermante, il est néanmoins recommandé d'inclure un slash (/) dans la balise afin de les repérer aisément. Ce type de balise est placé précisement là où l'on veut afficher l'élément.
      Par exemple :

      • <img />
      • <br />
      • <div />

      (Source : Votre première page web en HTML - OpenClassrooms)

  • - Les attributs

    Un attribut fait office d'option pour une balise et permet de préciser un effet que l'on veut obtenir.
    Par exemple : 

    <a href="https://www.w3schools.com">This is a link</a> où a est la balise pour un lien html et href est l'attribut.

    (Source : HTML Attributes - W3Schools)

  • - La différence entre inline et block

    Une balise block sera toujours affichée sur une nouvelle ligne et occupera toute la largeur disponible, tandis qu'à l'inverse une balise inline restera sur sa ligne actuelle et n'occupera que la largeur nécessaire.

    • Exemples inline :

      • <article></article>
      • <blockquote></blockquote>
      • <hr>

    • Exemples block :

      • <acronym></acronym>
      • <br>
      • <div></div>

    (Source : HTML Block and Inline Elements - W3Schools)

  • - Le markdown et l'HTML

    • En markdown
      • Insérer une image : 
        ![alt text](https://github.com/adam-p/markdown-here/raw/master/src/common/images/icon48.png "Logo Title Text 1")
      • Insérer une citation: 
         This is a quote.
      • Insérer une vidéo youtube : 
        [![IMAGE ALT TEXT HERE](http://img.youtube.com/vi/YOUTUBE_VIDEO_ID_HERE/0.jpg)](http://www.youtube.com/watch?v=YOUTUBE_VIDEO_ID_HERE)
      (Source : Markdown Cheatsheet · adam-p/markdown-here Wiki)

    • En HTML
      • Insérer une image : <img src="[URL de l'image]" alt="[Texte alternatif]">
      • Insérer une citation: <q></q>
      • Insérer une vidéo youtube : <iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY"></iframe>
      (Source : HTML Tutorial - W3Schools)
